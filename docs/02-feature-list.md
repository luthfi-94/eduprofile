# Feature Overview

1. Public Website
2. Admin Dashboard

---

# User Roles

The application currently has one primary role.

## Administrator

Permissions:

- Login
- Manage Website Settings
- Manage School Profile
- Manage Principal
- Manage Teachers
- Manage Facilities
- Manage News
- Manage News Categories
- Manage Albums
- Manage Gallery
- Manage PPDB Information
- Manage Contact Information
- Manage Users
- Logout

---

## Public Visitor

Permissions:

- View Home Page
- View School Profile
- View Principal Welcome
- View Teachers
- View Facilities
- View News
- View Gallery
- View PPDB Information
- View Contact Information

Visitors do not require authentication.
